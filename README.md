# ⚽ FIFA 2026: Offensive vs Defensive Analytics

## 📌 Description
This project provides an exploratory analysis of FIFA World Cup 2026 team statistics.  
The goal is to compare **offensive performance** (goals, assists, shots) and **defensive performance** (goals conceded, goalkeeper saves, discipline) to highlight different playing styles and team balance.
